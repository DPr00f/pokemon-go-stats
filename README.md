# Example app

(Here you can find the app running on an heroku server)[http://pokeiv.herokuapp.com/]

# Install

```git clone git@github.com:DPr00f/pokemon-go-stats.git```

```cd pokemon-go-stats```

```npm install```

# Run
`npm start` and navigate your browser to `http://127.0.0.1:3000`
